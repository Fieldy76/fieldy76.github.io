HTML Table Reference

Form Tags

Tag Name Description
<form> Form Used to create an HTML form for user input.
<input> Input Specifies an input field where the user can enter data.
<textarea> Text Area Defines a multi-line text input control.
<button> Button Defines a clickable button.
<select> Select Used to create a drop-down list.
<option> Option Defines an option in a select list.
<optgroup> Option Group Used to group related options in a <select> element (drop-down list).
<fieldset> Field Set Used to group related elements in a form.
<label> Label Defines a label for several elements.
<output> Output Used to represent the result of a calculation (like one performed by a script).

Form Attributes

Attribute Name Description
accept-charset Accept Charset Specifies the character encodings that are to be used for the form submission.
action Action Specifies where to send the form-data when a form is submitted.
autocomplete Auto Complete Specifies whether a form should have autocomplete on or off.
enctype Encoded Type Specifies how the form-data should be encoded when submitting it to the server (only for method="post").
method Method Specifies the HTTP method to use when sending form-data.
name Name Specifies the name of a form.
novalidate No Validate Specifies that the form should not be validated when submitted.
rel Relationship Specifies the relationship between a linked resource and the current document.
target Target Specifies where to display the response that is received after submitting the form.
